# Mario & Luigi mini-game in Javascript

This repository contains the source code for the reproduction of the mini-game *Boats of Revival* from *Mario & Luigi: Bowser's Inside Story* on *Nintendo DS*.

## Context

This game was made by Gabriel Antoine as part of the "*UE : DEV. APP. : Programmation web avancée*" course of *Haute École de la Province de Liège*.
